# SaniZ 사용법

화면을 확대하고 그 위에 그림 · 글씨를 쓰는 도구예요. (Windows 전용)
`SaniZ.exe`를 더블클릭하면 트레이에 상주하고, 아래 단축키로 바로 씁니다.

<br>

## 🔑 모드 켜고 끄기

| 키 | 기능 |
|----|------|
| `Alt + 1` | **확대 모드** 켜기 / 끄기 |
| `Alt + 2` | **판서 모드** 켜기 / 끄기 |
| `Esc` | 종료 |

<br>

## 🔍 확대 모드 (`Alt + 1`)

- **마우스 이동** → 커서를 따라 화면이 확대되며 이동
- **마우스 휠** → 확대 배율 조절
- **좌클릭** → 그 화면 그대로 캡처되며 **판서 모드로 자동 전환** (멈춘 화면에 바로 그리기)

<br>

## ✏️ 판서 모드 (`Alt + 2`)

**그리기**

| 입력 | 기능 |
|------|------|
| 드래그 | 자유 곡선 |
| `Ctrl` + 드래그 | 사각형 |
| `Shift` + 드래그 | 직선 |

**색 · 텍스트 · 지우개 · 배경**

| 키 | 기능 |
|----|------|
| `R` / `G` / `B` / `Y` | 펜 색 (빨강 / 초록 / 파랑 / 노랑) |
| `T` | 텍스트 입력 (누른 뒤 위치 클릭 → 입력, `Shift+Enter` 줄바꿈, `Enter` 완료) |
| `E` | 지우개 켜기 / 끄기 (휠로 크기 조절) |
| `K` / `W` | 어두운 / 밝은 배경 |

<br>

## ⚙️ 설정

트레이 아이콘 **더블클릭** → 설정 창.
단축키, 확대 배율, 펜 굵기, 지우개 크기, 텍스트 글꼴, 펜 · 배경 색을 바꿀 수 있어요.
바꾼 설정은 자동으로 저장됩니다.

<br>

> 💡 확대 · 판서를 켠 순간의 화면이 "정지"됩니다. 안 쓸 때는 트레이에 조용히 있으면서 메모리를 거의 쓰지 않아요.
